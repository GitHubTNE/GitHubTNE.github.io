Cryptocurrency Trading Bot

Identify winning crypto projects before they gain widespread attention

Project Overview

This project, developed in early 2024, was designed to identify the current 'meta' theme in crypto to catch winning coins early. By automating this process, it successfully turned a profit by discovering and evaluating new projects before they gained widespread attention.

Key Features

Automated Scanning

Scans newly listed or trending coins on decentralized exchange trackers

URL Collection

Collects external website URLs from coin pages for analysis

AI Classification

Uses OpenAI's GPT to score URLs based on specified keywords or themes

User-Friendly GUI

Simple interface for both technical and non-technical users

Notifications

Telegram notifications for winning projects

Tools and Libraries

customtkinter

Beautiful Soup

Telegram API

Code Breakdown

The implementation includes a coin finder tool that automates scanning, collects external links, classifies websites using GPT, and provides a user-friendly GUI. The code is modular with clear separation between classification logic and the GUI functionality.
